# 🧩 PokeGuessr

This project is a small fan-made game inspired by the classic “Who’s That Pokémon?” concept. The goal is to guess a Pokémon based on its silhouette.

This project is intended for:
- Learning and experimentation (SwiftUI, iOS development, API usage, UI/UX)
- Demonstrating app architecture and game mechanics
- Personal enjoyment

## 📡 Data Source

This app uses the public API:
- https://pokeapi.co/

Pokémon data (including images and descriptions) is fetched at runtime and is not redistributed or bundled with the application.
